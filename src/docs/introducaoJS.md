JAVASCRIPT - Não tem nada ver com JAVA

JAVASCRIPT - É case sensitive (maiusculo e minusculo se diferem)
JAVASCRIPT - Não é necessário tipar variáveis (adicionar tipo ex: string nome, int idade)
JAVASCRIPT - Não é preciso de compilador apra executar os scripts
